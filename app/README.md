# App Directory (Next.js App Router Root)

This directory forms the root of the Next.js 15 App Router configuration:

- `layout.tsx` - Initial default layouts loading font preloads and styling overrides.
- `page.tsx` - Interactive mockup previewing the Citizen, Driver, and Admin layouts.
- `globals.css` - Stylesheet resolving Tailwind CSS v4 and keyframe configs.
- `(citizen)/` - Route group reserved for citizen portal pages.
- `(driver)/` - Route group reserved for driver vehicle portal pages.
- `(admin)/` - Route group reserved for municipal admin pages.
- `(auth)/` - Route group reserved for login/signup pages.
- `(landing)/` - Route group reserved for public landing/marketing pages.
