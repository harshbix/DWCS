# Dashboard Portal Views Directory

This directory contains modular portal component partitions representing the main roles of the EcoCollect platform:

- `citizen-portal.tsx` - Layout and tabs dedicated to standard residents (billing summaries, QR presentation, reporting sensors, live telemetry).
- `driver-portal.tsx` - Compact layout for collectors (stop routes toggles, scanner checks, breakdown reporting forms).
- `admin-portal.tsx` - Full administrative layout (KPI dashboard cards, Weekly efficiency bar charts, citizen directories search grids, live map telemetry cards, dispatch audits).
