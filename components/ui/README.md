# UI Components Directory

This directory contains atomic user interface elements following standard accessibility and layout structures:

- `button.tsx` - Focusable button with loading indicator spinner variants.
- `input.tsx` - Accessible text input text field with validation error indicators.
- `card.tsx` - Layout cards for panels, headers, stats, and text containers.
- `badge.tsx` - Visual tags for Paid/Overdue, low/high, pending/resolved labels.
- `avatar.tsx` - User profile circle image loader with character fallback.
- `table.tsx` - Tabular grids for payments list and administration tables.
- `dialog.tsx` - React Portals modal panels with overlay fades.
- `dropdown.tsx` - Context menus and selection dropdown triggers.
- `tabs.tsx` - Component menu toggle wrappers.
