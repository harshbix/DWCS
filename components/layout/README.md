# Layout Components Directory

This directory contains layout wrapper shells composing primary route views:

- `top-navigation.tsx` - The system header bar containing flag badges, live notifications dropdowns, and development role selectors.
- `sidebar.tsx` - Collapsible sidebar displaying Overview, Citizens, Maps, and Settings for administrative users.
- `dashboard-layout.tsx` - The core application shell that mounts headers, conditional sidebars, and main content wrappers based on active roles.
- `page-container.tsx` - Animated layout bounds that contain margins, maximum width grids, and padding parameters.
