# EcoCollect Tanzania - Documentation Hub

Welcome to the definitive source of truth for the **EcoCollect Tanzania** project. This directory contains comprehensive documentation covering every aspect of the system, from frontend architecture to database design, security audits, and deployment guides.

## Project Summary
**Status:** ✅ Implemented | 🟡 Release Candidate
EcoCollect Tanzania is a modern, real-time Municipal Solid Waste Collection platform. It bridges the gap between citizens, waste collection logistics, and municipal authorities by providing transparent billing, live GPS tracking, and route optimization. 

## Architecture Overview
The system is built on a decoupled **Next.js 15 (React)** frontend using **App Router**, backed by a powerful **Supabase (PostgreSQL)** backend that handles Auth, Database, Storage, and Realtime sync. An auxiliary **Express.js API** resides in the `/backend` directory for specific secure logic operations (e.g., payment gateways).
