# Stores Directory

This directory contains client-side State Management stores built using **Zustand**:

- `theme.store.ts` - Visual light/dark styling settings.
- `sidebar.store.ts` - Collapse and toggle sidebar visibility state.
- `auth.store.ts` - Temporary mock user context and layout switching roles (citizen, driver, admin).
- `notification.store.ts` - Client alert notifications array and read tracker.
- `loading.store.ts` - App-wide global blocking progress indicators.
