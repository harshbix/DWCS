# Motion Directory

This directory contains Framer Motion presets and animation variant definitions to prevent animation code duplication:

- `shared.ts` - Shared spring presets and cubic easing variables matching the Stitch design.
- `page-transition.ts` - Standard layout and route transitions.
- `fade.ts` - Opacity reveals.
- `slide.ts` - Off-canvas sidebars, sliding panels, and drawer selectors.
- `scale.ts` - Micro-scale animations for hover states, button taps, and pop revealing elements.
- `modal.ts` - Screen backdrops and center-floating dialog wrappers.
- `stagger.ts` - Stagger orchestration variants for list cascades and loading cards.
